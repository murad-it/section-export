# Bitrix Catalog Section Export / Import

Скрипт для переноса раздела каталога вместе со всеми вложенными разделами и товарами с одного сайта на **1C-Bitrix** на другой.

Решение реализует сервер-серверную синхронизацию каталога через **POST-запросы** без использования REST API, только средствами ядра Bitrix.

---

# Возможности

- перенос **раздела каталога**
- перенос **всех вложенных разделов**
- перенос **товаров внутри разделов**
- сохранение **структуры дерева разделов**
- перенос **основных данных товаров**
- перенос **цен**
- перенос **изображений (preview)**
- пакетная отправка данных
- использование **только API ядра Bitrix**
